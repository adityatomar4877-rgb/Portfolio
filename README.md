# Aditya Tomar - Portfolio

Welcome to the source code for my personal portfolio! This project showcases my skills, experience, and recent work.

## 🚀 Live Demo

**[View Live Portfolio](https://portfolio-omega-two-epg8g3sy3l.vercel.app/#)** 


## 🛠️ Tech Stack

- **Framework:** [React](https://react.dev/) + [Vite](https://vitejs.dev/)
- **Routing:** [TanStack Router](https://tanstack.com/router)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/) + Custom CSS
- **Components:** [Radix UI](https://www.radix-ui.com/)
- **Deployment:** [Vercel](https://vercel.com/)

## 💻 Running Locally

To run this project on your local machine:

1. Clone the repository:
   ```bash
   git clone https://github.com/adityatomar4877-rgb/Portfolio.git
   ```
2. Navigate to the project directory:
   ```bash
   cd portfolio
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
